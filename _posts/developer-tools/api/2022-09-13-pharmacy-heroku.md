# 1 &nbsp; 폴더 만들기
---

로컬디스크C에 폴더 하나 `HTML_SERVER_TEST` 만듭시다\
그리고 windows + R을 눌러 CMD 입력해준후 이 경로로 옵시다\
`cd ..` 입력하면 경로 하나씩 줄어듭니다\

```console
C:\HTML_SERVER_TEST> 
```
이 경로로 맞춘 후 `npm init` 입력합시다.\
그러고 쭉 엔터하다가 마지막에 \
`Is this OK? (yes)` 이거 나오면 y 입력하고 엔터칩시다
