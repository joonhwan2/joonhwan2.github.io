## git rebase 
---

## 잠깐 ❗

> 어딘가에 push로 내보낸 작업은  git rebase 하면안됨 ---> 왜냐?\
> 하는 순간 가지치기처럼 내용이 병합되면서 원하고자 하는 내용과 달라짐.\
> 로컬저장소(내 노트북)에서 push 하기 전 작업은 rebase 해도 무방하다
